# Проект: Realco

### Описание проекта

Проект сделан на основе макета Figma для устройств с разрешениями до 320px. Проект представляет собой шаблон сервиса для поиска недвижимости.

### Используемые технологии

* Адаптивная верстка
* Grid и flex
* Медиа-запросы
* JavaScript
* Работа с DOM
